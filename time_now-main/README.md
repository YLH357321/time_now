## 功能概述

Time.html 是一个美观的动态时间显示网页，具有以下功能：

### 核心功能
1. **实时时钟**: 显示当前时间（24小时制，精确到秒）
2. **日期显示**: 显示公历日期（年月日）
3. **农历显示**: 自动计算并显示农历日期（含干支纪年）
4. **星期显示**: 显示当前星期
5. **时区信息**: 显示时区偏移（UTC+8）
6. **倒计时功能**: 四个可配置的倒计时卡片
7. **名言弹窗**: 点击时间区域显示激励名言
8. **流星动画**: 四张图片随机生成的流星效果

### 视觉特效
- 玻璃拟态（Glassmorphism）设计风格
- 动态渐变背景（紫色调）
- 流星动画效果（从边缘滑过）
- 悬浮动画效果
- 响应式布局（适配不同屏幕尺寸）
- 精美的卡片悬停效果

## 流星动画特效

### 特性说明
- **流星大小**: 35px × 35px
- **生成频率**: 每 0.3-1 秒生成一颗流星
- **旋转速度**: 360-1080 度（快速旋转）
- **动画时长**: 2-5 秒
- **移动方向**: 从左侧或顶部边缘滑向对侧
- **透明度**: 0.8（半透明效果）

### 流星图片
项目包含 4 张流星图片：
- `3F3F3E.webp`
- `3F3F3F.webp`
- `3F3F3F3F.webp`
- `3F3F3F3E3F.webp`

### 动画效果
- 流星从屏幕边缘随机位置生成
- 向对侧边缘滑过并旋转
- 淡入淡出效果
- 自动消失和清理 DOM 元素

## 配置修改指南

### 1. 修改倒计时日期

倒计时日期在 JavaScript 代码的 `updateCountdown()` 函数中定义。

#### 四个倒计时的配置

**倒计时1：距离攀登&卓越还有（2027年1月1日）**
```javascript
const competitionDate = new Date(2027, 0, 1);
```

**倒计时2：距离那一天……（2026年6月9日）**
```javascript
const physicsDate = new Date(2026, 5, 9);
```

**倒计时3：距离第43届复赛（2026年9月19日）**
```javascript
const competition43Date = new Date(2026, 8, 19);
```

**倒计时4：距离刷完物理学难题集萃（2026年5月1日）**
```javascript
const physicsCompleteDate = new Date(2026, 4, 1);
```

**注意**：JavaScript 中月份从 0 开始（0=1月，11=12月）

#### 修改倒计时标题

在 HTML 部分修改显示的文字（约第 550-580 行）

### 2. 修改流星动画参数

在 JavaScript 代码的 `createMeteor()` 和 `startMeteors()` 函数中：

#### 调整流星大小
```css
.meteor {
    width: 35px;  /* 修改这个值 */
    height: 35px; /* 修改这个值 */
}
```

#### 调整生成频率
```javascript
const interval = 300 + Math.random() * 700; // 300-1000 毫秒
```

#### 调整旋转速度
```javascript
const rotation = Math.random() * 720 + 360; // 360-1080 度
```

#### 调整动画时长
```javascript
const duration = 2 + Math.random() * 3; // 2-5 秒
```

#### 添加/修改流星图片
```javascript
const meteorImages = [
    'your-image-1.webp',
    'your-image-2.webp',
    // 添加更多图片...
];
```

### 3. 修改名言内容

在 HTML 部分（约第 600 行）：
```html
<p class="modal-text">你的名言内容</p>
```

### 4. 修改颜色主题

在 CSS 中修改主容器背景（约第 65 行）：
```css
.main-container {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 50%, #f093fb 100%);
}
```

## 使用方法

### 本地使用
1. 双击 `time.html` 文件，使用默认浏览器打开
2. 或右键选择"打开方式"，选择指定浏览器

### 查看名言
点击中间的时间区域，会弹出名言框

### 关闭弹窗
- 点击右上角的 × 按钮
- 点击弹窗外部区域
- 按 ESC 键

## 技术架构

### HTML 结构
- **主容器**: `.main-container`（16:9 比例）
- **流星容器**: `.meteor-container`（承载流星动画）
- **布局系统**: 三列布局（左、中、右）
- **卡片组件**: 5个玻璃拟态卡片
- **弹窗组件**: 模态对话框

### CSS 特性
- Flexbox 布局
- CSS 动画（pulse, glow, meteor-fall）
- 玻璃拟态效果（backdrop-filter）
- 响应式媒体查询
- 渐变色和阴影

### JavaScript 功能
- **时间更新**: `setInterval` 每秒刷新
- **农历计算**: 内置农历算法（1900-2100年）
- **倒计时计算**: 日期差值计算
- **流星生成**: 随机位置和参数
- **Web Animations API**: 流星移动和旋转
- **事件处理**: 点击、键盘事件

## 常见问题

### Q: 时间显示不正确
A: 检查系统时间设置，网页自动读取本地系统时间。

### Q: 倒计时显示为0
A: 检查目标日期是否已过期，或者代码中的日期格式是否正确。

### Q: 农历显示错误
A: 农历算法支持1900-2100年，超出此范围可能不准确。

### Q: 流星不显示
A: 检查浏览器控制台（F12）查看错误信息，确保图片文件路径正确。

### Q: 样式显示异常
A: 确保使用现代浏览器，清除浏览器缓存后重试。

### Q: 如何添加更多流星图片
A: 将图片放入 time_now 文件夹，然后在 `meteorImages` 数组中添加文件名。

### Q: 如何修改更新频率
A: 修改 `startMeteors()` 函数中的 `interval` 值（单位：毫秒）。

### Q: 如何固定在屏幕上
A: 使用浏览器的全屏模式（F11），或将浏览器设置为全屏应用。
